# Dierk Thomas Archive

A structured archive of source documents, timelines, and reference materials relating to Dierk Thomas and associated business history.

## Overview

This repository is intended to preserve relevant records in a clear, searchable, and source-based format.

The focus is on documentation, chronology, and traceability. Materials included here may consist of public records, correspondence, filings, historical business documents, and other reference material that helps establish context over time.

## What this repository contains

- Source documents in PDF format
- Timelines and factual summaries
- Indexed references to relevant events, entities, and records
- Supporting notes based on underlying documents

## Repository structure

- `docs/`  
  Timelines, summaries, indexes, and supporting write-ups

- `pdfs/`  
  Source documents and archived records

- `README.md`  
  Repository overview and documentation principles

## Documentation approach

This archive is built around a simple principle:

**primary sources first**

Where possible, each entry should include:

- title
- date
- source
- short description
- relevance
- status note where needed, for example: public record, correspondence, allegation, dispute, or confirmed finding

## Important note

The presence of a document in this repository does not by itself establish guilt, liability, or a final legal conclusion unless that is explicitly stated in the underlying source.

Readers should review original documents directly and distinguish between:

- verified records
- allegations
- disputed claims
- legal findings
- commentary or interpretation

## Purpose

This repository exists to:

- preserve records that may otherwise become difficult to locate
- organise information in a transparent way
- allow independent review of source material
- maintain a documented chronology of relevant events

## Corrections

If a document is mislabelled, incomplete, incorrectly dated, or should not be included, corrections supported by evidence should be considered.

## Disclaimer

This repository is maintained for documentation and archival purposes. It does not constitute legal advice.
